# Ouroboros

---

Cursor based pagination for Ecto. Thanks to [duffelhq](https://github.com/duffelhq) for the [paginator](https://github.com/duffelhq/paginator) library, from which this library grew.

---

## Installation

```elixir
def deps do
  [
    {:ouroboros, "~> 0.1.0"}
  ]
end
```
